Window.alert

Esse "alerta" alguma coisa na tela, apenas mostrando uma mensagem e pedindo para continuar

Window.confirm

Já esse mostra alguma coisa em uma janela assim como o alert, porem apenas com a possibilidade de confirmar algo

Window.prompt

Esse aqui é mais interassante, pois ele recebe um input vindo do usuario para a janela, podendo até mesmo ser usado como o valor de uma variavel
var nomeVariavel = window.prompt("Aqui vai o valor da variavél "nomeVariavel" na qual o usuario irá definir qual será")
